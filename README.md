## Files

- `index.html` - main portfolio page
- `styles.css` - responsive page styling
- `script.js` - renders report cards and live Power BI preview sections
- `data/reports.js` - report titles, summaries, skills, insights, and Power BI URLs

## How to Customize

Edit `data/reports.js` to update:

- report titles
- summaries
- tools used
- insights
- Power BI links

Edit the footer in `index.html` to add your name, LinkedIn, GitHub, and email.

## GitHub Pages

1. Push these files to your GitHub repository.
2. Go to `Settings` > `Pages`.
3. Choose `Deploy from a branch`.
4. Select the `main` branch and `/root`.
5. Open the GitHub Pages URL generated by GitHub.

## Note About Power BI Embeds

The page includes embedded Power BI previews. Some Power BI links may require Microsoft sign-in or may not load inside an iframe depending on tenant settings. Each report also has a direct "Open in Power BI" button, which is the most reliable option for resume reviewers.
